# Kannanware Design System

## Color Palette (DARK PREMIUM THEME)
- Background: #0c0c0c (near-black), bg2: #121212
- Foreground: #f2f2f2 (near-white for text)
- Gold: #E8A000 (primary accent, used for headings, accents, CTAs)
- Card: #141414
- Secondary: #1f1f1f
- Border: #262626
- Muted foreground: #8c8c8c

## Typography
- Font: Ubuntu (all headings and body)
- Headings: clamp-based responsive sizing, tracking -0.03em
- Eyebrow pattern: uppercase, tracking .2em, gold color, with ::before line

## Custom Cursor
- K logo PNG cursor following mouse with smooth lag (0.14 factor)
- Scale up on hover over interactive elements
- Hidden on mobile (cursor: auto)
